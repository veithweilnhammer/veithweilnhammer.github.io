# Stage 3 Review: Gemini

## 1. Style-guide violations

**Violation:** Punchy announcement sentence.
> "The harder problem sits behind that case."
*Fix:* Combine with the following sentence to remove the dramatic beat. "Behind this single vivid case lies a harder problem: a system in conversation with tens of millions of people..."

**Violation:** Punchy announcement sentence.
> "The two kinds of outcome leave very different records."
*Fix:* Merge into the explanation. "A harmful exchange that precedes a suicide and a prevented harm leave very different records: a harmful exchange produces a legible trail..."

**Violation:** Sentence fragment for effect / antithesis.
> "A prevented harm produces an absence."
*Fix:* Integrate into the surrounding sentence to state the fact plainly. "A harmful exchange produces a legible trail—transcripts, a date, a named person—while a prevented harm leaves no record."

**Violation:** Rhetorical crescendo / evocative metaphor.
> "The ledger is lopsided in what it records, and public judgment settles on the column it can read."
*Fix:* State the consequence plainly. "Because caused harms are documented and prevented harms are not, public judgment naturally focuses on the tragedies."

**Violation:** Punchy setup sentences / short dramatic cadence.
> "Medicine has been answering that kind of question for decades, and it has built tools for it. Population screening is the clearest example."
*Fix:* Combine into a single, measured sentence. "Medicine has long answered questions like this using tools built for population screening."

**Violation:** "Not X, it is Y" construction.
> "None of this argues for giving up on the arithmetic; it argues for the opposite, because the alternative to counting is not neutrality."
*Fix:* State the positive claim directly. "These difficulties make explicit calculation more necessary, because failing to measure trade-offs means the decision is driven entirely by the most visible harm."

**Violation:** Punchy setup / antithesis.
> "Honest accounting is easy to state and hard to carry out."
*Fix:* Delete this throat-clearing sentence and start the paragraph with the substantive requirement. "Honest accounting requires measuring the benefit side rather than assuming it is zero..."
